### 2_변수와 타입들
 *	자바스크립트에는 변수 타입이 없다(데이터 타입은 있음)
 *	var : 단순하게 변수를 선언한다 (생략 가능)
 * let : 변경이 허용된 변수를 선언한다
 * const : 변경이 금지된 변수를 선언한다
